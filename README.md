# 🪙 Conversor de Monedas - Challenge Alura Latam

¡Bienvenido al Conversor de Monedas! Este proyecto es parte del desafío de formación en Java de Alura Latam. Es una aplicación de consola que consume tasas de cambio en tiempo real para ofrecer conversiones precisas entre distintas divisas.

## 🚀 Características
- **Consumo de API Real:** Utiliza [ExchangeRate-API](https://www.exchangerate-api.com/) para obtener datos actualizados.
- **Interfaz de Consola:** Menú interactivo para el usuario.
- **Soporte Multimoneda:** Conversiones entre Dólar (USD), Peso Argentino (ARS), Real Brasileño (BRL) y Peso Colombiano (COP).
- **Manejo de Errores:** Control de excepciones para fallos de red o ingresos de datos inválidos.

## 🛠️ Tecnologías Utilizadas
- **Java 17+**
- **Gson (Google):** Para el mapeo de objetos JSON.
- **HttpClient:** Para las solicitudes asíncronas a la API.
- **Records de Java:** Para una estructura de datos más limpia.
